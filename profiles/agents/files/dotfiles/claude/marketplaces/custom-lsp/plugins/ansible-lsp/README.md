# ansible-lsp

Ansible language server for Claude Code, providing playbook validation, module documentation, and code intelligence.

## Supported Extensions
`.yml`, `.yaml` (when detected as Ansible content)

## Installation

Install via npm:

```bash
npm install -g @ansible/ansible-language-server
```

Or with Homebrew:

```bash
brew install ansible-language-server
```
