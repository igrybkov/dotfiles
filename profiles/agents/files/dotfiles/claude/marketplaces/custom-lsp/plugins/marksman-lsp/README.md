# marksman-lsp

Markdown language server for Claude Code with wiki-link support, go-to-definition, find-references, and rename across markdown files.

## Supported Extensions
`.md`

## Features
- Wiki-link (`[[link]]`) navigation (Zettelkasten/Obsidian style)
- Go to definition across markdown files
- Find references
- Document symbols
- Workspace symbols

## Installation

Install via Homebrew:

```bash
brew install marksman
```
